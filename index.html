Обо мне

Привет, меня зовут Олег! Я Fullstack-разработчик на Python. Закончил курсы по вёрстке и Java Script в онлайн-школе Skillbox. Сейчас я изучаю курс по Type Script, а в ближайших планах освоить курс по фреймворку Vue.js. Для меня очень важен профессиональный рост и развитие.

В своих работах я использую адаптивную вёрстку, семантические теги, простые и удобные в использовании элементы.

Мне интересно читать статьи на Хабре, смотреть видео о новых технологиях во фронтенд-разработке, играть на досуге с друзьями в настольные игры и активно отдыхать на природе.

Мои работы

  • Сайт туристической компании

    Это мой первый сайт по вёрстке. Я использовал адаптивную вёрстку карточек с помощью свойства flex. Для интуитивно понятного восприятия кнопок и ссылок применил псевдоэлементы focus, hover, active. Изображения в карточках сделал фоном с помощью отдельных селекторов в CSS. В разметке сайта использовал семантические теги. После изучения анимации подключил сторонюю библиотеку для демонстрации движения заголовка. Позже реализовал бургерное меню для экранов мобильных устройств.

  • Сайт художественной галереи Blanchard

    По завершении курса веб-вёрстки я сделал дипломную работу. Здесь я применил технологию flex-box для удобного расположения элементов на веб-странице, также использовал медиа-запросы для адаптации сайта под различные экраны устройств. В разделе "Галерея" кастомизировал чек-боксы для фильтрации работ. В разделе "Галерея" настроил несколько источников для оптимизации загрузки изображения в необходимом разрешении в зависимости от размера экрана устройства.
