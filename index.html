Обо мне

Привет, меня зовут Олег! Я Fullstack-разработчик на Python. Закончил курсы по вёрстке и Java Script в онлайн-школе Skillbox. Сейчас я изучаю курс по Type Script, а в ближайших планах освоить курс по фреймворку Vue.js.

Мои интересы: читаю статьи на Хабре, смотрю видео про новые технологии во фронтенде, на досуге играю с друзьями в настольные игры и люблю активный отдых на природе.
