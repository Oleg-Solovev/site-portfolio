Обо мне

Привет, меня зовут Олег! Я - frontend-разработчик сайтов. Закончил курсы по вёрстке и Java Script в онлайн-школе Skillbox. В ближайших планах изучить Type Script и фреймворк Vue.js. Для меня очень важен профессиональный рост и развитие.

В своих работах я использую современные технологии адаптивной вёрстки: Flexbox, Grid, BEM.

С интересом читаю статьи на Хабре и смотрю видео о новых технологиях по frontend-разработке и не только.

На досуге я катаюсь на велосипеде, лыжах и коньках, хожу в бассейн, играю с друзьями в настольные игры, люблю активный отдых на природе.

Стек, который я использую:

  • HTML
  • CSS
  • JavaScript
  • jQuery
  • Git

  • CRM клиентская база

    Это CRM-система управления клиентской базой в которой реализованы функции добавления, изменения и удаления клиентов из браузерного хранилища данных "LockalStorage". Взаимодействие с интерфейсом осуществляется с помощью модальных окон. В карточке пользователя можно добавлять, редактировать и удалять контактные данные. В таблице клиентов есть возможность сортировать данные по колонкам. После каждого изменения происходит рендеринг таблицы клиентов. Для оптимизации кода и работы с DOM элементами я использовал библиотеку jQuery.
